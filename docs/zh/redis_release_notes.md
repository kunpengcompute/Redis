#### 更新说明
##### Redis网络异步优化
通过KrAIO调度Poll机制优化，提升在同物理机部署proxy、Redis的场景下本地网络交互性能。redis-benchmark通过网卡IP对本地Redis 7.0.15单实例压测set、get命令在2C2T场景下性能提升5%。
