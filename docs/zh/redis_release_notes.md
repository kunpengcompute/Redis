### v26.0.RC1版本说明
#### 更新说明
##### Redis sockmap优化

**新增功能：**

新增Redis sockmap优化特性，在Redis的localhost访问场景中，sockmap能够显著降低网络协议栈的开销，带来显著的性能提升。
